Introdução à Manipulação do DOM em JavaScript

O DOM (Document Object Model) é uma interface de programação para documentos HTML,
XML e SVG. Ele representa a estrutura do documento como um conjunto de objetos,
permitindo que os programadores manipulem sua estrutura, estilo e conteúdo
dinamicamente por meio de linguagens de script como JavaScript. A manipulação do DOM
é essencial para criar interações dinâmicas e atualizações em páginas da web.

O que é Manipulação do DOM?

A manipulação do DOM refere-se ao processo de acessar, modificar e interagir com
elementos HTML em uma página da web. Isso inclui selecionar elementos, alterar seu
conteúdo, atributos, estilos e responder a eventos do usuário.

Principais aspectos da Manipulação do DOM

  1. Selecionar elementos HTML:
    • Métodos como 'getElementById', 'getElementsByClassName', 'getElementsByTagName' e
      'querySelector' são usados para selecionar elementos com base em seus IDs,
      classes, tags ou seletores CSS.
    • Exemplo: 
      •             var elemento = document.getElementById('meuElemento');
  2. Modificar conteúdo, atributos e estilos:
    • Uma vez que os elementos são selecionados, você pode modificar seu conteúdo
      usando propriedades como 'innerHTML', 'innerText', 'textContent', etc.
    • Os atributos podem ser alterados usando métodos como 'setAttribute' e 'removeAttribute'.
    • Os estilos podem ser modificados diretamente através da propriedade 'style'.
    • Exemplo: 
      •             var elemento = document.getElementById('meuElemento');
            elemento.innerHTML = 'Novo conteúdo';
            elemento.setAttribute('class', 'destaque');
            elemento.style.color = 'red';
  3. Manipulação de eventos:
    • Eventos como cliques, passagens do mouse, pressionamentos de tecla, etc., podem
      ser manipulados atribuindo funções a elementos HTML usando 'addEventListener'.
    • Exemplo: 
      •             var botao = document.getElementById('meuBotao');
            bota o.addEventListener('click', function() {
                alert('O botão foi clicado!');
            });

Esses são os principais aspectos da manipulação do DOM em JavaScript, permitindo aos
desenvolvedores criar páginas da web interativas e dinâmicas.
