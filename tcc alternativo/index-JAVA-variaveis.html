Variáveis e Tipos de Dados

Declaração de Variáveis

A declaração de variáveis é um conceito fundamental em programação, permitindo que
você armazene e manipule dados. Em muitas linguagens de programação, a declaração
de variáveis envolve escolher um nome descritivo e associar a variável a um tipo de
dado específico. Vamos considerar um exemplo em JavaScript: 

 // Declaração de variáveis
 let idade; // Declaração de uma variável chamada 'idade'
 idade = 25; // Atribuição de valor à variável 'idade'
 console.log(idade); // Saída: 25

Neste exemplo, criamos uma variável chamada 'idade' e atribuímos o valor 25 a ela.

Tipo de Dados

  • Números

    • Em programação, os números podem ser inteiros ou decimais. Variáveis do tipo
    número são utilizadas para armazenar valores matemáticos. Exemplos: 

    let idade = 25;      // Número inteiro
let altura = 1.75;   // Número decimal
  • Strings

    • Strings são sequências de caracteres, como texto. Elas são comumente usadas
    para representar palavras, frases ou qualquer combinação de caracteres
    alfanuméricos. Exemplos: 

    let nome = "Ana";        // String com um nome
let saudacao = "Olá!";   // String com uma saudação
  • Booleanos

    • Variáveis booleanas têm apenas dois valores possíveis: true ou false. Elas
    são frequentemente usadas para representar condições lógicas. Exemplos: 

    let estaChovendo = false;   // Indica se está chovendo ou não
let temSol = true;          // Indica se há sol ou não
  • Arrays

    • Arrays são estruturas de dados que armazenam coleções ordenadas de
    elementos. Cada elemento pode ser de um tipo diferente, permitindo a criação
    de listas de dados. Exemplos: 

    let frutas = ["maçã", "banana", "laranja"];    // Array de strings
let numerosPrimos = [2, 3, 5, 7, 11];           // Array de números inteiros
  • Objetos

    • Objetos são estruturas de dados que permitem agrupar informações relacionadas
    em pares chave-valor. Cada propriedade do objeto tem um nome exclusivo. Exemplos: 

    let pessoa = {
    nome: "Carlos",
    idade: 28,
    cidade: "Rio de Janeiro"
};   // Objeto representando informações de uma pessoa

Esses tipos de dados formam a base para a construção de programas em diversas
linguagens de programação. Ao compreender esses conceitos, você estará apto a
manipular e organizar dados de maneira eficaz em suas aplicações.
