Funções em Programação: Uma Introdução Abrangente

As funções são blocos de código reutilizáveis que realizam uma tarefa específica
dentro de um programa. Elas desempenham um papel crucial na organização e
modularização do código, tornando-o mais legível, eficiente e fácil de manter. Vamos
explorar mais sobre funções, incluindo como declará-las, seus parâmetros e argumentos,
e como funcionam os retornos.

1. Declaração de Funções

Em muitas linguagens de programação, incluindo Python, JavaScript e C++, a sintaxe
básica para declarar uma função é a seguinte: 

    function nomeDaFuncao(parametros) {
         "Corpo da função"
         "Código a ser executado"
        return resultado;
    }

Por exemplo, uma função simples em JavaScript que adiciona dois números e retorna o
resultado pode ser declarada da seguinte forma: 

    function soma(a, b) {
        return a + b;
    }

2. Parâmetros e Argumentos de Função

Os parâmetros são variáveis ​​declaradas na definição da função e servem como
marcadores de posição para os valores que serão passados para a função quando ela
for chamada. Os argumentos são os valores reais passados para a função durante a
chamada.

Por exemplo, na função 'soma(a, b)', 'a' e 'b' são parâmetros. Ao chamar essa função e
passar valores específicos para 'a' e 'b', esses valores são os argumentos: 

    var resultado = soma(3, 5);

Neste caso, '3' e '5' são os argumentos passados para os parâmetros 'a' e 'b', respectivamente.

3. Retorno de Funções

As funções podem retornar valores utilizando a palavra-chave 'return'. O valor
retornado pode ser qualquer tipo de dado, como números, strings, listas, ou até
mesmo objetos mais complexos.

Por exemplo, uma função em JavaScript que verifica se um número é par pode ser
definida da seguinte forma: 

    function ehPar(numero) {
        if (numero % 2 === 0) {
            return true;
        } else {
            return false;
        }
    }

Quando chamamos essa função e passamos um número como argumento, ela retorna true
se o número for par e false caso contrário: 

    var resultado = ehPar(4);  ""Retorna true"

Exemplos Adicionais

    function calcularMedia(listaNumeros) {
        var total = listaNumeros.reduce((acc, curr) => acc + curr, 0);
        var media = total / listaNumeros.length;
        return media;
    }
    
    var notas = [7, 8, 9, 6, 8];
    var mediaNotas = calcularMedia(notas);
                    
    function saudacao(nome) {
        return "Olá, " + nome + "!";
    }
    
    var mensagem = saudacao("Ana"); " Retorna 'Olá, Ana!'"

Em resumo, as funções são blocos de código encapsulados que realizam tarefas
específicas e podem ser chamadas repetidamente em um programa. Elas ajudam a
tornar o código mais modular, facilitando a manutenção e promovendo a reutilização.
