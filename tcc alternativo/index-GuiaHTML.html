Html

Guia Basico do HTML

HTML (HyperText Markup Language) é uma linguagem de marcação usada para estruturar
o conteúdo de páginas web. Ele consiste em elementos, cada um representando uma
parte da página, como títulos, parágrafos, listas, links, imagens, etc.

Vamos ao Aprendizado ?

Estrutura Básica

O esqueleto básico de um documento HTML inclui as tags <html>, <head>, <titl>,
<body>, e outros elementos opcionais. A tag <!DOCTYPE html> define a versão do
HTML. 

     <!DOCTYPE html>
     <html lang="pt-br">
     <head>
          <meta charset="UTF-8">
          <meta name="viewport" content="width=device-width, initial-scale=1.0">
          <title>Título da Página</title>
     </head>
     <body>
     <!-- Conteúdo da Página -->
     </body>
     </html>
  • Função <!DOCTYPE html>

    • Declara a versão do HTML sendo utilizada.

  • Função <html lang="pt-br">

    • Define o idioma da página.

  • Função <meta charset="UTF-8">

    • Especifica o conjunto de caracteres UTF-8 para suportar diversos idiomas.

  • Função <meta name="viewport" content="width=device-width, initial-scale=1.0">

    • Configura a escala inicial e largura da tela para dispositivos móveis.

Títulos e Parágrafos

Títulos

As tags de título <h1> a <h6> são usadas para definir os níveis de cabeçalho,
indicando a hierarquia do conteúdo. 

                        <h1>Este é um Título de Nível 1</h1>
                        <h2>Este é um Título de Nível 2</h2>
                        <!-- ... Até h6 -->

Cada nível de título indica uma importância relativa do conteúdo, com <h1> sendo
o mais importante e <h6> o menos importante.

Parágrafos

A tag <p> é utilizada para definir parágrafos no HTML. 

                        <p>Isto é um parágrafo.</p>
                        <p>Outro parágrafo aqui.</p>

A tag <p> é usada para envolver o texto de cada parágrafo, facilitando a
formatação e leitura do conteúdo.

Listas

Lista Não Ordenada

A tag <ul> é usada para criar listas não ordenadas, e <li> para cada item da lista. 

                        <ul>
                            <li>Item 1</li>>
                            <li>Item 2</li>>
                            <li>Item 3</li>>
                        </ul>
  • Função ul

    • indica o início de uma lista não ordenada.

  • Função li

    • define cada item na lista.

Lista Ordenada

A tag <ol> é usada para criar listas ordenadas. 

                        <ol>
                           <li>Primeiro Item</li>
                           <li>Segundo Item</li>
                           <li>Terceiro Item</li>
                        <ol>
  • Função ol

    • indica o início de uma lista ordenada.

  • Função li

    • define cada item na lista.

A principal diferença é que as listas ordenadas exibem números ou letras para
indicar a ordem, enquanto as listas não ordenadas utilizam marcadores.

Links

A tag <a> cria links. O atributo href especifica o destino do link.

    <a href="https://www.exemplo.com">Texto do Link</a>
  • Função Href

    • contém o URL para o qual o link aponta.

Imagens

A tag <img> é usada para incorporar imagens. O atributo src especifica o caminho
da imagem e alt fornece uma descrição alternativa. 

            <img src="caminho/para/imagem.jpg" alt="Descrição da Imagem">
  • Função src

    • define o caminho da imagem.

  • Função alt

    • fornece um texto alternativo para a imagem, importante para acessibilidade.

Formulários

A tag <form> é usada para criar formulários interativos. Os elementos <label>,
>input>, e <button> são frequentemente utilizados. 

            <form action="/processar" method="post">
                <label for="nome">Nome:</label>
                <input type="text" id="nome" name="nome"><br>
            
                <label for="email">Email:</label>
                <input type="email" id="email" name="email"><br>
            
                <input type="submit" value="Enviar">
            </form>
  • Função action

    • especifica o URL para o qual os dados do formulário serão enviados.

  • Função method

    • indica o método de envio, geralmente 'get' ou 'post'.

Estilos

Estilo Inline

Você pode aplicar estilos diretamente a um elemento usando o atributo style. 

<p style="color: red; font-size: 16px;">Este parágrafo possui estilos inline.</p>

O atributo style contém as regras de estilo CSS, como cor e tamanho da fonte.

Folha de Estilo Externa

A tag <link> é usada para vincular uma folha de estilo externa ao documento HTML. 

                        <link rel="stylesheet" type="text/css" href="estilos.css">
  • Função rel="stylesheet"

    • indica que o link é para uma folha de estilo.

  • Função type="text/css

    • especifica o tipo de arquivo.

  • Função href

    • contém o caminho para a folha de estilo externa.

Tabelas

A tag <table> é usada para criar tabelas. As tabelas consistem em linhas (<tr>),
colunas (<td> para células de dados) e cabeçalhos (<th> para cabeçalhos de
coluna ou linha). 

                    <table>
                        
                            <th>Header 1</th>
                            <th>Header 2</th>
                        </tr>
                        <tr>
                            <td>Dado 1</td>
                            <td>Dado 2</td>
                        >/tr<
                    </table>
  • Função <table>

    • define a tabela.

  • Função <tr>
    • define uma linha.
  • Função <th>
    • define um cabeçalho (opcional).
  • Função <td>
    • define uma célula de dados.

Comentários

Comentários no HTML são úteis para documentar o código. Eles não são exibidos na
página, mas podem ser visualizados no código-fonte. 

                    <!-- Este é um comentário HTML -->

Comentários começam com <!-- e terminam com -->.

Elementos Semânticos

Elementos semânticos fornecem significado ao conteúdo. Alguns exemplos incluem
<header>, <nav>, <article>, <section>, <footer>. Eles ajudam a estruturar a página
de maneira mais compreensível para os desenvolvedores e navegadores. 

                    <header>
                      <h1>Título do Cabeçalho</h1>
                    </header>

Elementos semânticos dão sentido ao conteúdo, facilitando a leitura e manutenção do código.

Vídeos e Áudio

As tags <video> e <audio> são usadas para incorporar conteúdo multimídia em uma página web. 

                    <video width="320" height="240" controls>
                      <source src="video.mp4" type="video/mp4">
                      Seu navegador não suporta o elemento de vídeo.
                    </video>
                  
                    <audio controls>
                      <source src="audio.mp3" type="audio/mp3">
                      Seu navegador não suporta o elemento de áudio.
                    </audio>

A tag<source> define a origem do conteúdo multimídia.
O atributo controls adiciona controles de reprodução.

Mapas de Imagem

A tag <map> junto com <area> é usada para criar mapas de imagem interativos. 

    <img src="plano.png" alt="Plano" usemap="#mapa">
    <map name="mapa">
        <area shape="rect" coords="0,0,50,50" href="area1.html" alt="Área 1">
        <area shape="circle" coords="100,100,50" href="area2.html" alt="Área 2">
    </map>
  • Função <map>

    • define a área do mapa de imagem.

  • Função <area>
    • define as áreas interativas no mapa.
