Trabalhando com Arrays e Objetos em JavaScript

Arrays

Arrays são estruturas de dados fundamentais em JavaScript que armazenam uma coleção
de elementos. Aqui estão alguns dos métodos mais comuns que você pode usar para
manipular arrays:

  • 'push( )'

    • O método 'push()' é usado para adicionar um ou mais elementos ao final de um
    array. Ele modifica o array original e retorna o novo comprimento do array.

    exemplo: 

    let array = [1, 2, 3];
array.push(4);
console.log(array); // Output: [1, 2, 3, 4]

    Neste exemplo, o número '4' é adicionado ao final do array 'array', aumentando
    seu comprimento para 4.

  • 'pop( )'

    • O método 'pop()' remove o último elemento de um array e retorna esse elemento.
    Ele também modifica o array original.

    exemplo: 

    let array = [1, 2, 3];
let removedElement = array.pop();
console.log(removedElement); // Output: 3
console.log(array); // Output: [1, 2]

    Neste caso, o número '3' é removido do final do array 'array', e é atribuído à
    variável 'removedElement'.

  • 'shift( )'

    • O método 'shift()' remove o primeiro elemento de um array e retorna esse elemento.
    Ele também modifica o array original, deslocando os índices dos elementos restantes.

    exemplo: 

    let array = [1, 2, 3];
let removedElement = array.shift();
console.log(removedElement); // Output: 1
console.log(array); // Output: [2, 3]

    Neste exemplo, o número '1' é removido do início do array 'array', e é atribuído
    à variável 'removedElement'.

  • 'unshift( )'

    • O método 'unshift()' é usado para adicionar um ou mais elementos ao início de um
    array. Ele modifica o array original e retorna o novo comprimento do array.

    exemplo: 

    let array = [2, 3];
array.unshift(1);
console.log(array); // Output: [1, 2, 3]

    Neste caso, o número '1' é adicionado ao início do array 'array', aumentando seu
    comprimento para 3.

  • 'map( )'

    • O método 'map()' cria um novo array com os resultados da chamada de uma função
    para cada elemento do array. Ele não modifica o array original.

    exemplo: 

    let array = [1, 2, 3];
let doubledArray = array.map(num => num * 2);
console.log(doubledArray); // Output: [2, 4, 6]

    Neste exemplo, a função de mapeamento dobra cada elemento do array original
    'array', criando um novo array 'doubledArray' com os resultados.

  • 'filter( )'

    • O método 'filter()' cria um novo array com todos os elementos que passam no
    teste implementado pela função fornecida. Ele não modifica o array original.

    exemplo: 

    let array = [1, 2, 3, 4, 5];
let filteredArray = array.filter(num => num % 2 === 0);
console.log(filteredArray); // Output: [2, 4]

    Neste exemplo, a função de filtro retorna 'true' apenas para os números pares,
    resultando em um novo array 'filteredArray' contendo apenas os números pares do
    array original 'array'.

Objetos

  • Adicionando Propriedades

    • Em JavaScript, você pode adicionar novas propriedades a um objeto simplesmente
    atribuindo um valor a uma nova chave.

    exemplo: 

    let person = { name: 'John', age: 30 };
person.job = 'Developer';
console.log(person); // Output: { name: 'John', age: 30, job: 'Developer' }

    Neste exemplo, adicionamos uma nova propriedade 'job' ao objeto 'person',
    atribuindo-lhe o valor ''Developer''.

  • Acessando Propriedades

    • Você pode acessar as propriedades de um objeto usando a notação de ponto
    (objeto.propriedade) ou a notação de colchetes (objeto['propriedade']).

    exemplo: 

    console.log(person.name); // Output: John

    Neste caso, estamos acessando a propriedade 'name' do objeto 'person'.

  • Removendo Propriedades

    • Você pode remover propriedades de um objeto usando o operador 'delete'. 

    delete person.age;
console.log(person); // Output: { name: 'John', job: 'Developer' }

    Neste exemplo, removemos a propriedade 'age' do objeto 'person'.

Espero que esses exemplos e explicações detalhadas tenham sido úteis para entender
como trabalhar com arrays e objetos em JavaScript!
