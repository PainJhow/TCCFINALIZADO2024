Atributos

Oque são Atributos do HTML ?

Atributos em HTML são características adicionais aplicadas
a elementos HTML para controlar seu comportamento, aparência
e funcionalidade em uma página web. Esses atributos fornecem
informações extras, permitindo desde a edição de conteúdo com
contenteditable até a definição de comportamentos específicos,
como arrastar elementos com draggable, garantindo assim maior
interatividade, acessibilidade e personalização na construção de
uma página web.

Como Funciona os Atributos ?

Os atributos em HTML são informações adicionais que podem ser incluídas
nos elementos para fornecer detalhes específicos sobre como esses elementos
devem se comportar ou ser exibidos. Cada atributo tem um propósito único e
pode ser usado para controlar diversos aspectos, desde a aparência visual
até o comportamento interativo de um elemento na página web.

Por exemplo, o atributo "href" em um elemento "a" é usado para especificar o
destino de um link, enquanto o atributo "src" em um elemento "img" define o
caminho da imagem a ser exibida na página.

Esses atributos são escritos dentro das tags dos elementos HTML e podem variar
dependendo do tipo de elemento. Eles desempenham um papel importante na estruturação
e funcionalidade das páginas da web, permitindo aos desenvolvedores controlar e
personalizar como o conteúdo é apresentado e interage com os usuários.

Alguns Atributos Exemplificados

  • Contenteditable

    • Esse atributo permite que o conteúdo de um elemento HTML seja editável diretamente
    na página, como se fosse um editor de texto simples. Ao adicionar
    contenteditable="true" a um elemento, como <div> ou <p>, o navegador permite que o usuário
    edite o texto dentro desse elemento. Isso pode ser útil para criar áreas de texto
    editáveis em aplicações web ou até mesmo para permitir a edição de conteúdo diretamente na página.

                                <div contenteditable="true">
                                {texto}
                                </div>
  • SpellCheck

    • Este atributo é aplicado a elementos como campos de texto ou áreas de texto e controla se o navegador deve verificar
    a ortografia do texto digitado. Adicionar spellcheck="true" habilitará a verificação ortográfica automática para o texto
    inserido nesses elementos.

                                <textarea spellcheck="true"></textarea>
  • ContextMenu

    • Ao definir esse atributo em um elemento, como <div> ou <p>, você pode associar um menu de contexto
    personalizado que será exibido ao clicar com o botão direito do mouse nesse elemento. Isso pode ser
    útil para fornecer opções específicas de contexto para o usuário.

                                <div contextmenu="menuID">Clique com o botão direito para ver o menu.</div>
                            <menu type="context" id="menuID">
                            <menuitem label="Opção 1"></menuitem>
                            <menuitem label="Opção 2"></menuitem>
                            </menu>
  • Translate

    • Esse atributo controla se o conteúdo de um elemento deve ser traduzido automaticamente pelo
    navegador. Define-se como translate="yes" para permitir a tradução automática ou translate="no"
    para impedir a tradução.

                                <p translate="no">Este texto não deve ser traduzido.</p>
  • Draggable

    • Quando aplicado a elementos HTML, como <div> ou <img>, o atributo draggable permite que eles sejam
    arrastados e soltos em outras áreas da página ou até mesmo em aplicativos externos.

                                <img src="imagem.png" draggable="true" alt="Imagem arrastável">
  • SandBox

    • Esse atributo é usado principalmente em elementos <iframe> para restringir as ações permitidas dentro
    do iframe, criando um ambiente "sandbox". Pode ser útil para executar conteúdo de terceiros com certas
    restrições de segurança.

                                <iframe src="paginaexterna.com" sandbox="allow-scripts"></iframe>
  • Hidden

    • Simplesmente oculta um elemento da página quando definido como "hidden". Isso pode ser útil para ocultar
    temporariamente um elemento sem removê-lo do DOM.

                                <p hidden>Este parágrafo está oculto.</p>
  • Scoped

    • Usado em elementos <style>, o atributo "scoped" limita o alcance das regras CSS apenas ao elemento
    pai. Isso evita que as regras de estilo se apliquem a outros elementos na página.

                                <style scoped>
                                p {
                                    color: red;
                                }
                            </style>
                            <p>Este texto será vermelho devido ao escopo do estilo.</p>
  • Nonce

    • É utilizado em <script> e <style> para garantir que apenas scripts e estilos autorizados sejam
    executados. O valor do atributo deve ser um número aleatório gerado pelo servidor para prevenir ataques de injeção
    de código.

                                <script src="script.js" nonce="nonce-gerado-pelo-servidor"></script>
  • IsMap

    • Este atributo é usado em <img> para criar imagens sensíveis a mapas. Ao definir "ismap", é possível
    associar coordenadas a um mapa de imagem para criar áreas clicáveis na imagem.

                                <img src="imagem.png" ismap usemap="#mapa">
                              <map name="mapa">
                               <area shape="rect" coords="0,0,50,50" href="pagina1.html">
                               <area shape="circle" coords="100,100,50" href="pagina2.html">
                              </map>

"draggable". Esses atributos não apenas aprimoraram a estética, mas também garantiram uma interação mais dinâmica, enriquecendo a
experiência do usuário neste site.
