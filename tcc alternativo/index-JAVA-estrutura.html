Estruturas de Controle de Fluxo em JavaScript

Estruturas de Controle de Fluxo em JavaScript são mecanismos que permitem direcionar o
fluxo de execução do código com base em condições específicas ou repetir blocos de
código enquanto uma condição é verdadeira.

If, else, else if, Switch case e Loops (for, while)

  • if, else, else if

    • O 'if' é como um farol que guia o caminho do nosso programa com base em condições. Se
    uma condição for verdadeira, uma ação é executada; caso contrário, podemos definir
    uma alternativa usando 'else'. Quando temos múltiplas condições, utilizamos 'else if'.
    Aqui está um exemplo: 

        let idade = 18;
    
    if (idade >= 18) {
        console.log("Você é maior de idade.");
    } else {
        console.log("Você é menor de idade.");
    }
  • Switch Case

    • O 'switch' é como um menu de opções, permitindo comparar uma variável com uma
    lista de valores e executar um bloco de código correspondente ao valor
    encontrado. Aqui está como usar em JavaScript: 

        let diaDaSemana = 3;
    let nomeDia;
    
    switch (diaDaSemana) {
        case 1:
            nomeDia = "Domingo";
            break;
        case 2:
            nomeDia = "Segunda-feira";
            break;
        case 3:
            nomeDia = "Terça-feira";
            break;
        default:
            nomeDia = "Dia inválido";
    }
    
    console.log("Hoje é " + nomeDia);
  • Loops (for, while)

    • Os loops são como máquinas incansáveis que executam tarefas repetidamente até
    que uma condição seja atendida. O loop 'for' é usado quando sabemos exatamente
    quantas vezes queremos repetir uma tarefa, enquanto o loop 'while' é usado quando
    queremos repetir uma tarefa até que uma condição seja falsa. Aqui estão exemplos
    em JavaScript: 

        // Loop for
    for (let i = 0; i < 5; i++) {
        console.log("Número:", i);
    }
    
    // Loop while
    let contador = 0;
    while (contador < 5) {
        console.log("Contagem:", contador);
        contador++;
    }
